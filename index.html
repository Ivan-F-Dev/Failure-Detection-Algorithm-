Расшифровка цвета датчиков:
1) голубой - при стартовой инициализации
2) красный - при высоком сигнале с датчика
3) серый - при низком сигнале с датчика
4) зеленый - с датчика приходит сигнал в пределах нормы
Формат ввода:
1) для обрыва ввести "номерУчастка + 0"
2) для К/З ввести "номерУчастка + 1"
Например, для моделирования К/З на 4-ом участке ввести "41"
